# ✨ AI 穿搭助手

一款专为女性设计的智能穿搭工具，解决"不知道怎么穿才好看"的烦恼。

---

## 🎯 解决什么问题

- **尺码拿不准**：网购衣服总是买大买小？输入身材数据和商品尺码表，帮你算出最适合的尺码
- **不知道自己适合什么**：不清楚自己的体型？不知道自己该避雷哪些面料和版型？AI 帮你分析
- **场合穿搭纠结**：通勤、约会、校园...不同场合该怎么搭配？按场景推荐，再也不用纠结
- **虚拟试穿**：上传照片和衣服图片，AI 生成上身效果，足不出户也能"试穿"
- **衣橱管理**：整理自己的衣橱，了解每件衣服的特点和搭配可能

---

## 📸 功能展示

| 功能 | 说明 |
|------|------|
| 身材档案 | 输入身高体重，选择体型标签，记录避雷面料和版型 |
| 虚拟试穿 | 上传人物照和衣服图，AI 生成上身效果图（需配置 API） |
| 尺码推荐 | 根据身材数据和商品尺码表，智能推荐合适尺码 |
| 场景穿搭 | 通勤、约会、校园、面试...按场合推荐适合的搭配 |
| AI 穿搭顾问 | 有穿搭问题随时问，像闺蜜一样给你建议 |
| 衣橱管理 | 记录和管理自己的衣服，快速筛选和搭配 |

---

## 💡 适用场景

- 🛒 **网购前**：不确定尺码或版型是否适合，先来问问 AI
- 👗 **出门前**：今天穿什么？根据场合和天气快速决策
- 👠 **买新衣时**：这件衣服适合我吗？适配度分析帮你做决定
- 🗄️ **整理衣橱**：给衣服打标签，发现更多搭配可能

## 🚀 开始使用
